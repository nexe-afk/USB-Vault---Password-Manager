# USB 密码保险箱 v1.0 - 使用说明书

USB 密码保险箱是一款基于“物理加密狗”理念设计的本地密码管理工具。它结合了硬件访问控制与高强度加密算法，确保您的敏感数据永远不会上传到云端。

---

## 🛡️ 核心安全特性

1. **零知识架构**：所有加密和解密过程均在本地浏览器中完成，服务器仅存储加密后的密文。
2. **AES-256-GCM 加密**：使用行业标准的对称加密算法保护每一条记录。
3. **PBKDF2 密钥派生**：主密码通过 100,000 次哈希迭代派生出加密密钥，有效抵御暴力破解。
4. **物理隔离模拟**：模拟 U 盘挂载逻辑，只有在“硬件”检测到且主密码正确时才能访问数据。
5. **内存安全**：提供“清除内存并锁定”功能，一键擦除内存中的敏感密钥。

---

## 🚀 快速使用指南

### 1. 初始化与解锁
*   **检测状态**：进入页面后，右上角会显示“USB 加密狗”状态。本演示版本默认模拟已连接。
*   **输入主密码**：在左侧“主身份验证”面板输入您的主密码（建议 8 位以上）。
*   **解锁保险箱**：点击“初始化解密”。系统将根据您的密码派生出唯一的加密密钥。

### 2. 管理密码记录
*   **添加记录**：解锁后，在右侧会出现“添加新的安全记录”表单。输入网站名称、用户名和密码，点击“保存至 USB 保险箱”。
*   **查看密码**：在“加密存储索引”列表中，点击“解密”按钮即可查看明文密码。
*   **复制密码**：点击解密后的密码旁边的复制图标，可快速将密码存入剪贴板。
*   **删除记录**：点击垃圾桶图标可永久删除该条加密记录。

### 3. 安全退出
*   **一键锁定**：点击左侧的“清除内存并锁定”按钮。这将立即销毁内存中的加密密钥并重置 UI 状态。

## 🛠️ 浏览器扩展开发（进阶）

本工具不仅是一个网页应用，还提供了完整的浏览器扩展集成方案。您可以切换到 **“扩展工具包”** 标签页查看相关代码：

1. **Native Messaging 宿主程序**：
    *   提供了一个 Python 脚本示例，用于在真实环境下读取物理 U 盘中的 `vault.db` 文件。
    *   负责与 Chrome 浏览器进行安全的标准流通信。
2. **Manifest V3 配置**：
    *   提供了 Chrome 扩展的配置文件模板，声明了必要的权限（`nativeMessaging`）。

## ⚠️ 注意事项

*   **请务必牢记主密码**：由于采用零知识架构，如果您丢失了主密码，没有任何人（包括开发者）能够找回您的数据。
*   **本地存储**：本演示版本的数据存储在项目根目录的 `vault.db` 中。
